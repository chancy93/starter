# 🚀 启步器

> 每次启动，都是一次胜利。

启步器是一个帮你克服「启动困难」的 AI 小工具。输入你一直拖延的大任务，AI 会自动拆解成 3-5 个微步骤，每个步骤用 5 秒倒计时帮你做好心理准备，拍照打卡确认完成。

---

## 为什么需要它？

写论文、整理房间、开始健身……这些事难的不是"怎么做"，而是 **"开始做"** 。

当脑子里只有"写论文"三个字时，它是一团模糊的阻力。但如果变成——
1. 打开文档编辑器
2. 写下标题和 3 个要点
3. 写第一段草稿（不追求完美）
4. 通读一遍，标记要改的地方

——每一步都变得清晰可执行。**降低启动门槛，而不是提高意志力。**

---

## 使用方式

1. 在浏览器中打开 `index.html`
2. 输入你想做但一直拖延的事
3. 点击 **🤖 AI 拆解**
4. 选一个最轻松的步骤，点 **🚀 启动**
5. 5 秒倒计时后拍照打卡，确认完成 ✅
6. 全部完成后解锁成就 🏆

支持添加到手机主屏幕（PWA），离线也能用。
